Orchestration Enablers ()
Definition: Orchestration Enablers ABB is a Grouping Component that refers to the components and practices that support the development and deployment of orchestration applications. Source: ETSI Network Functions Virtualisation Standard Source reference: https://portal.etsi.org/NFV/NFV_White_Paper2.pdf Last modification: 2024-01-28 Identifier: http://data.europa.eu/dr8/OrchestrationEnablersGrouping LOST view: Technical view - application EIRA concept: eira:ArchitectureBuildingBlock ABB name: eira:OrchestrationEnablersGrouping Interoperability Saliency: The Orchestration Enablers Grouping ABB is salient for interoperability because it contains both, application services and components, that can enable the deployment and development of orchestration services.
