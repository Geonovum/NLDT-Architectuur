Linked Open Data ()
Definition: Linked Open Data ABB is a Representation that enables the procedures of publishing structured data enriched with metadata to ensure data understanding, connection, findability and link with informational resources. Source: EUROPEANA PRO Source reference: https://pro.europeana.eu/page/linked-open-data Last modification: 2024-01-28 Identifier: http://data.europa.eu/dr8/LinkedOpenDataRepresentation LOST view: Semantic view EIRA concept: eira:ArchitectureBuildingBlock ABB name: eira:LinkedOpenDataRepresentation Example: The following implementation is an example of how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): EuropeanaPro https://pro.europeana.eu/page/linked-open-data Interoperability Saliency: The Linked Open Data Representation ABB is salient for semantic interoperability because it provides a set of publishing, sharing, and connection methods that allow linking and publishing the data from digital public services, making it more meaningful and increasing the quality and scope of data and services. Moreover, it increases data discoverability and maximises the reuse and exchange of data.
